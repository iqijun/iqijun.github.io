力哥七步定投策略

Posted on 2017-10-31 In 理财小知识 Word count in article: 1.3k Reading time ≈ 5 mins.
  1. 4大原则：
    1. 长期坚持定期不定额投资
    2. 长期坚持止盈不止损投资
    3. 只问贵贱，不看趋势
    4. 越跌越买，月涨越卖
  2. 5中投资策略
    1. 全天候配置策略
    2. 低估值择时轮动策略
    3. 价值平均策略
    4. 网格交易策略
    5. 动态再平衡策略
  3. 7个具体步骤
    1. 完成风险偏好测试
    2. 全面统计家庭财务信息，搞清楚有多少钱可以投资于金融资产
    3. 做好家庭资产配置方案
      1. 提取生活紧急备用金（3-6个月的生活支出放到货基中） ---现金流规划
      2. 购置适合的保险的产品： --- 家庭保障规划
        1. 消费险
        2. 储蓄型保险
      3. 将钱分为长期和短期（3年内） --- 消费规划和储蓄规划
    4. 设计存量资金的定投计划
      1. 设计定投阶段
      2. 布局定投阶段
      3. 维持定投阶段
      4. 存量资金定投计划策略
        1. 经济景气时积极进取，不景气时保守
        2. 经济周期
          1. 复苏阶段： 经济上行， 通胀（CPI）下行 增持股票
          2. 经济过热阶段： 经济上行，通胀上行 增持股票，大宗商品，减持债券、现金
          3. 滞涨阶段： 经济下行，通胀上行
            1. 减持各类资产，现金为王
          4. 衰退阶段： 经济下行，通胀下行
            1. 增持债券 美林时钟投资理论，按经济周期投资
        3. 指数估值
          1. PE 市盈率
          2. PB 市净率
          3. 绝对估值：市盈率超过50倍的不碰
          4. 相对估值：以自身的估值作为依据
          5. 市盈率分位点低于40%时开始定投，同时参考市净率的分位点
        4. 几个重要
          1. 存量资金定投布局要在一年内完成，资金量较少的建议每月定投一次，钱越多定投频率越高。
          2. 更推荐定投场内指数基金
    5. 在一年内完成定投布局 1. 低位多投，高位少投 2. 价值平均策略：每月完成定投后的市值是目标市值，而不是每月定投固定资金（目标市值恒定），例如：计划每月定投1万月，当第二个月定投时，持有基金的市值已达到了1万2千元，则第二月的定投金额改为8000，使市值达到2万月即可，如果第二月定投时市值降为8000元，则第二月的定投金额改为1万2千元。如果第二月定投时，市值已达到2万1千元，则要卖掉1千元市值的基金 3. 提高定投额标准 PE分位点 < 20% ,基准定投额提高1.5 PE分位点 < 15% ,基准定投额提高2 PE分位点 < 10% ,基准定投额提高2.5 PE分位点 < 5% ,基准定投额提高3 4. 减少定投额，抛售定投基金 
        当PE的分位点到达40即合理估值的区间时就要考虑减少定投或停止定投了。
    6. 对提前完成定投的基金进行网格收割（维持定投的操作）
      1. 网格交易操作
        1. 名词说明
        2. 网格交易的两个缺点：
          1. 熊市中可能买无可买
          2. 如果遇到大牛市，可能提前卖光
        3. 适用于震荡行情（与二八轮动恰恰相反）
      2. 当定投市值达到预定目标时，可以利用网格交易法则进行操作
        1. 从历史回测来看，当单边的熊市或牛市中网格可以放大一点
        2. 震荡行情中，网格就应该设置的小一点
        3. 最大的网格设置是20%，最小的网格设置是3%
        4. 上涨时的收割网格设置的要比下跌的时的补仓网格设置的要大一些，从而获得更大的利润空间
    7. 设置增量资金定投计划（要以年为周期）
      1. 全天候配置策略+低估值择时策略

    合理的资产配置+坚持科学的定投策略=永恒不变的投资王道
