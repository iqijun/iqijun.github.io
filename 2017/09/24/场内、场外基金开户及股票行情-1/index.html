场内、场外基金开户及股票行情

Posted on 2017-09-24 Edited on 2017-10-08 In 理财小知识 Word count in article: 1.3k Reading time ≈ 5 mins.

一、场外基金

  1. 不能在交易所上市交易的基金
  2. 直销、代销
  3. 直销：到基金公司官网注册开户，费用较低
  4. 代销：天天基金网，蛋卷基金
  5. 分红方式：现金分红、红利再投资。
  6. 货币基金统一都是红利再投资方式

二、 场内基金

  1. 手续费低、灵活、有一些套利玩法，无法设置定投
  2. 券商开户一般会开两个账户，一个是证券账户：沪A账户/深A账户；另一个是资金账户：用来对接银行存管账户。
  3. 银证转账，一般是在交易日的8:00-16:00
  4. 场内基金的买卖操作，要在券商交易系统中股票栏中操作，如果想尽快成交那么在卖出时选择买一价，买入时尽量选择卖一价。
  5. 场内基金分红方式默认现金分红且不能修改

三、股票行情软件

  1. K线图,又称作日本蜡烛线，其中实心的线叫做阴线，国内一般用绿色表示（海外用红色），空心的叫做阳线，国内用红色表示（为来显示效果好看，股票软件中有时会用实线表示，海外用绿色），所以有叫做的阴阳线；
  2. K线上方的细线叫做上影线，下方的细线叫做下引线
  3. K线中记录了开盘价，收盘价，当日最高价、当日最低价;当收盘价高于开盘价时，为阳线，上影线的最高点表示当日到达的最高价，下影线最低表示当日的最低价。
  4. 几种常见的K线
    1. 如果当天的开盘价或者收盘价就是当天的最高价那么就没有上影线，俗称光头
    2. 如果当天的开盘价或者收盘价时当天的最低价那么就没有下影线，俗称光脚
    3. 如果开盘价和收盘价相等那么柱子就会变成一条横线，加上上下影线，就是个十字型，叫做十字星，如果上下影线都很长，那么就叫做大十字星，反之叫做小十字星，如果只有下影线没有上影线就叫做T型光头的十字星，反过来，如果没有下影线只有上影线那么就叫做 倒T型光头十字星
    4. 如果当天开盘价，收盘价相等，并且没有什么波动（没有上影线和下影线）那么就叫做一字线
  5. 涨跌幅波动
    1. 小幅波动：涨跌幅0~1.5%
    2. 中幅破洞：涨跌幅1.5%~5%
    3. 大幅波动： 涨跌幅5%~10%
  6. K线的复权
    1. 向前复权：保持现有价格不变，将以前的价格折算响应缩减
    2. 向后复权：保持先前的价格不变，将除权后价格增加修复回去
  7. 移动平均线，简称均线（moving average，MA），如MA10就是指包含当天收盘价在内的20天的收盘价的平均值。
    1. 日均线指标：MA5，MA10 短期均线指标
    2. 季均线指标：MA30，MA60中期均线指标
    3. 年均线指标：MA120，MA250 长期均线指标
    4. 一般MA250就是年线
    5. 也可以自己设定均线。
    6. 短期均线超过中长期均线叫做金叉，意味着趋势向上，反之则叫做死叉
  9. 成交量：在k线图和均线图下方的柱状图
    1. 成交量均线（mavol）
    2. 技术分析研究股票的量、价、时、空
    3. 成交量决定了股票的流动性
  10. 涨幅：收盘价相对于前一个交易日收盘价的上涨幅度
  11. 振幅：当天最高价和最低价的差额相对于昨天的收盘价的百分比，体现的是股票或基金的活跃性
  12. 分时走势：黄色的线表示成交价格，白色的线表示成交量。
  13. 快捷键：
    1. F5切换分时走势和技术分析
    2. F6查看自选股
    3. F3快速查看上证指数
    4. F4快速查看深证成指
  14. 基金小白要避免的误区
    1. 把握时机，不能为了长期投资而长期投资，要学会止损，止盈
    2. 净值不是越便宜越好
      1. 净值低原因1.这只基金是新基金
      2. 本身业绩就差
      3. 老基金刻意通过大幅分红和拆分降低净值，有可能会摊薄老基金持有人的利益
      4. 是不是分红越多的基金越好？当然不是
