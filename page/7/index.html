  1. Found of Founds
  2. 替你排兵布阵进行资产配置获得最优的投资回报
  3. 不会受到传统基金的限制，比如股票型基金要买80%以上的股票，债券基金要买80%以上的债券
  4. 分类：内部管理+内部配置，外部管理+内部配置，内部管理+全市场配置，外部管理+全市场配置

  8. 投资单支基金的比例最高为20%
  9. 不能投资其他FOF基金
  10. 不得对FOF基金双重收费
  11. FOF不得投资分级基金
  12. 不得持续运作小于1年，最近定期报告披露规模小于2亿元的基金
  13. 不得收取投资了自家产品部分的管理费，托管费和销售费，此条规定有可能引发道德风险，投资自家基金产品的基金经理有可能会进行利益输送

    function getFileUrl(sourceId) {
    var url;
    if (navigator.userAgent.indexOf(“MSIE”)>=1) { // IE
    url = document.getElementById(sourceId).value;
    } else if(navigator.userAgent.indexOf(“Firefox”)>0) { // Firefox
    url = window.URL.createObjectURL(document.getElementById(sourceId).files.item(0));
    } else if(navigator.userAgent.indexOf(“Chrome”)>0) { // Chrome
    url = window.URL.createObjectURL(document.getElementById(sourceId).files.item(0));
    }
    return url;
    }

/**

    function preImg(sourceId, targetId) {
    var url = getFileUrl(sourceId);
    var imgPre = document.getElementById(targetId);
    imgPre.src = url;
    }

  1. 目标期限基金：前期高风险向低风险过渡当达到指定目标日期时风险最低。
  2. 目标风险基金：按目标风险分，分为激进、稳健、保守

二、 定增基金：

  1. 定增：定向增发，是指上市公司向符合条件少数投资者非公开发行股份的行为。
  2. 上市和增发的区别：
  3. 二次募资三种方式：
    1. 配股：按照原股东的持股比例，由原股东购买新发股票，配股完成后要进行除权操作，即将新老股票的价格平均；
      1. 带有一点强制性，老股东必须购买，如果不购买在除权操作后，老股东手的资金缩水。
      2. 如果老股东选择在配股前抛售手中的股票，就容易造成股价下跌
    2. 公开增发：直接在二级市场发售新股，容易被恶意买入
    3. 定向增发：由指定的购买者购买，可以是法人，投资机构，自然人
    4. 为防止利益输送，定向增发必须服从股东大会的决议；涉及外资的定增必须向国务院相关部门提交申请；对定增股价打折的要求：一般不低于定价基准日前20天股价平均值的90%；会有一年的限售期；控股股东，实际控制人及其控制企业认购的股份，境外战略投资者限售期要延长至3年；定增股东解禁首年不能减持大于50%，连续90天竞价减持不超过总股本1%；连续90日内大宗交易减持不超过2%，且受让方6个月内不能减持。
    5. 定增风险：
      1. 优质项目不够多
      2. 折价空间不够大，解禁后不确定性大
    6. 定增投资技巧
      1. 选择时机：适合在熊市投资
      2. 由于定增市场火热 定增价高于发行价

混合型基金和股票型基金

Posted on 2017-06-18 In 理财小知识 Word count in article: 1.5k Reading time ≈ 5 mins.
  1. 混合基金分类 见基金理财一课通
  2. 投资的风险：
    1. 非系统性风险:只有特殊情况下的某个投资发生风险，系统整体安全；在股市上选股风险就是非系统性风险就
    2. 系统性风险：系统整体发生风险，如熊市
  3. 风险对冲
    1. 定义： 投资两种不同种类的投资物，来避免风险。
  4. 混合型基金由于可以投资不同种类的投资品比如：货币基金、债券基金、股票基金因此可以形成风险对冲，使投资风险降低。
  5. 混合型基金的不确定性

    例如：灵活配置型基金的基金经理可以任意配置股票，债券的比例，因此可以能会由于基金经理的误判引发踏空或被套的风险。另外，由于投资者无法判断混合型基金投资种类的占比，无法预估风险，可能产生焦虑

  6. 股票型基金

    1. 股票型基金是公募基金四大品种中风险最大的
    2. 主动型股票基金和被动型股票基金（指数基金）
  7. 主动型股票基金的挑选技巧
    1. 挑选大型，知名的基金公司；21加超过1000亿的基金公司 天弘基金、工银瑞信、易方达、博时基金、招商基金、南方基金、华夏基金、嘉实基金、建信基金、中银基金、汇添富基金、鹏华基金、广发基金、银华基金、华安基金、华宝兴业、兴全基金、兴业基金、上投摩根、大成基金
    2. 选择好的基金经理
      1. 基金经理的基本背景，年龄、履历等
      2. 基金经理的操盘风格：持仓时间长短，看基金的持仓股票有哪些，基金换手率，对行业，主题的偏好
      3. 晨星投资风格箱
      4. 价值型股票和成长型股票：即白马股和黑马股;
        1. 白马股：业绩稳定，分红能力强； 蓝筹股
        2. 黑马股：业绩一般，分红少，成长空间大
      5. 价值型风格基金：
      6. 成长型风格基金：
    3. 挑选业绩好的：
      1. β系数：基金和业绩评价基准之间的相关性。业绩评价基准是指基金KPI，根基基金自身的特性定制的考核标准。
      基金类型 业绩评价基准
      货币基金 定期存款利率
      股票型基金 大盘走势
      指数型基金 所跟踪的指数

      主动性股票的业绩评价基准比较复杂，比如以两个不同的指数综合作为基准。
      β系数其实就是对基金所承受的系统风险做的量化表达。通俗来说就是收益弹性有多大，弹性越大，当基准涨的时候该基金涨的就越多，跌的时候同理。
      一般来说一个基金或股票的β细说在0.5~1.5之间波动。

      1. 与β系数相关的投资策略
    1. 高β策略：也叫高波策略。即选择β系数比较高的基金。适合与激进型的投资者，可以获得较高的收益但同时也会有较高的风险。也适合于在牛市中赚取高额回报
    2. 低β策略：也叫低波策略。与高波策略相反，适合保守型的投资者，同时在熊市中低β策略的基金比较抗跌。
      1. α系数：代表着基金的绝对回报或者叫做超额收益
        1. 基金的真实收益减去市场的无风险收益就是这个基金的风险收益。
        2. 示例：某基金的β系数是1.2，但某一时期内它的真实收益是基准收益的1.5倍，那么这个基金的α系数= 1.5-1.2=0.3，那么这0.3（30%）就叫做超额收益；但如果α系数为-30%，那么就叫做超额损失。
        3. α系数是靠基金经理的择时或选股的高超水平来决定的。α系数越高，基金经理越牛。
      2. R平方：用来计算α系数和β系数的精准度。R平方越高α、β系数的可信度就越高。R平方为100则代表可靠，如果是0，就是不靠谱；
      3. 夏普指数：评价基金的重要指标；该指数越高代表基金越值得投资。

        夏普比率= （基金回报率-市场无风险回报率）/标准差

      4. 特雷诺指数：反应基金承受单位系统风险所获得的超额收益。指数越高获得的超额收益越高。

        特雷诺指数 =（基金回报率-市场无风险回报率）/β系数

      5. 詹森指数：

    4. 选择规模适中的基金

      1. 如果规模过大，容易产生投资风格漂移。规模太小，容易遭遇流动性风险。
        投资种类 规模大小
        中小盘 10~20亿
        大盘 30~100亿
    5. 选择成本低的基金 管理费 1%~2% 一般在1.5% 申购费/赎回费： 申购费一般在1.5% ；赎回费一般在0.5%，一般持有时间越长赎回费率越低

  1. 混合基金分类 见基金理财一课通
  2. 投资的风险：
    1. 非系统性风险:只有特殊情况下的某个投资发生风险，系统整体安全；在股市上选股风险就是非系统性风险就
    2. 系统性风险：系统整体发生风险，如熊市
  3. 风险对冲
    1. 定义： 投资两种不同种类的投资物，来避免风险。
  4. 混合型基金由于可以投资不同种类的投资品比如：货币基金、债券基金、股票基金因此可以形成风险对冲，使投资风险降低。
  5. 混合型基金的不确定性

    例如：灵活配置型基金的基金经理可以任意配置股票，债券的比例，因此可以能会由于基金经理的误判引发踏空或被套的风险。另外，由于投资者无法判断混合型基金投资种类的占比，无法预估风险，可能产生焦虑

  6. 股票型基金

    1. 股票型基金是公募基金四大品种中风险最大的
    2. 主动型股票基金和被动型股票基金（指数基金）

一级债基的风险也是不同的，不同的一级债基规定的新股可以上市交易后持有股票的时间是不同的，持有时间越长风险越大。

特别注意：名字中带有纯债字眼的基金未必是纯债基金，也有可能是可转债基金。
画重点，债基名称带有ABC等小跟班的意义

如果只有AB两种，那么带有A的债券型基金是要收取申赎费用的;B类是不另外收取申赎费用的,而是把这些费用和托管费，管理费等折算到基金净值中预先扣除了这部分费用（销售服务费），一般每年收取0.2%~0.6%

如果分成ABC三类的债券基金，那么A类债券基金收取申购费（前端收费），B类收取赎回费，大部分公司持有时间越长，赎回费率越低，持有足够时间后可以免除赎回费。C类是收取销售服务费的。

  1. 定期开放式债券基金（定开债）
    1. 定义： 定期开放支持投资者申购赎回，开发期过后封闭运行
    2. 投资定开债的注意事项
      1. 选择定开债中纯债基金
      2. 别选择杠杆率太高的基金（160%）
      3. 封闭期不要太短，也不要太长，1年左右
    3. 直接投资债券的缺点
    4. 品种交少，投资门槛高
    5. 流动性差
    6. 投资性价比较低，因为相较于投资债券基金，只投资一个债券品种风险较高。
    7. 投资过于复杂，
  2. 基金分红
    1. 现金分红
    2. 红利再投资
  3. 基金累计净值
    1. 单位净值：通俗的讲单位净值就是交易日每天收市以后，基金持有的股票，权证，债券，现金，票据等在当天的价值之和除以当天的基金总份额，就得出当天的单位净值。比如某一天资产总值2亿，当天的基金总份额一亿，则单位净值为2元。一般来说单位净值是每天波动的，因为资产总值和总份额都是随机波动的
    2. 累计净值：是该基金自发行成立以来的净值，包括了分红，拆分份额等。
    3. 如果一支基金自发行以来，没有分过红，拆分份额，单位净值等于累计净值。否则，一般累计净值大于单位净值
  4. 债券指数基金
    1. 指数基金的优点
      1. 被动投资
      2. 投资成本低
      3. 简单明了
    2. 现有股票指数基金1000多个，债券指数基金20多个
    3. 不鼓励投资债券指数基金
