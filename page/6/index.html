把自己当成商品销售出去

Posted on 2017-11-30 In 杂谈 | 读过的书 Word count in article: 888 Reading time ≈ 3 mins.
最近“不务正业”地读了一本关于销售的书-- 诺瓦尔·霍金斯所著的《销售圣经》。书中提到了销售的三个要素，四个阶段。按我自己理解把书中提到这几点总结为人、知识、“术”；

销售的“人”

将销售的书中提到的最关键的人当然是销售人员。怎么成为一个优秀的销售人员？第一步拥有一个健康的身体。是的，身体是革命的本钱，做任何事情都是以身体能够自由行动为前提的。再进一步，有了健康的身体之后还要拥有一个令人愉悦的形象，否则在接近目标客户的时候肯定是会被拒在千里之外的。然后还有一颗勇敢的心，只有足够勇敢才能面对各种“蛮不讲理”的客户

销售的“知识”

你能指望一个没有一点计算机知识的人写出一套完美的代码吗？当然不可能。任何一个行当都有一定的入行门槛的，这就是入行所需要的知识。做销售要掌握销售的基本知识，还要对销售的东西有充分的了解，对专业知识的掌握程度给顾客的印象就是这个够不够专业，这个销售所在的公司够不够专业

销售的“术”

这是我之所以看这本书的原因的吧。我就是想从理论上认识一下平时是怎么被各种销售忽悠的。不过作者讲的就很上档次，完全不是忽悠这个层面了。总的来讲就是从情感和头脑两个方面攻破顾客。从情感上让顾客对产品产生兴趣，然后让顾客的头脑在战胜不想购买的“理性思维”。从而促使顾客购买产品。

把自己卖出去

混迹职场是以出卖时间的方式把自己“卖”出去，那怎么把自己卖个好价钱呢。我感觉可以套用销售中提到的三个方面来打造“自己”这款产品。

良好的形象

任何人都会喜欢俊男靓女，良好的形象会给面试官一个了完美的第一印象，于是便有了一个美好的开端。当然不能仅仅关注美丽的容颜，更要有革命的本钱–健康的身体。猝死事件还是时有发生，所以即使不为工作也应加强锻炼。

能够完成任务的知识

公司存在的目的就是盈利，雇佣员工就是为了完成盈利目标，那么要想成为一家公司的雇员当然要拥有为公司盈利的知识。作为雇员你的知识都是为了公司的利润，所以如果你掌握了能让公司盈利的知识，对公司来说就会有较大的吸引力，让公司觉得你够专业。

把自己推销出去技巧

把自己的才能展示出来，吸引公司的注意力，让公司明白你有能力完成公司的任务；这就如同销售人员在情感上吸引顾客，让顾客对产品产生足够的兴趣。其次，还要在理性上让公司想吸收你这个员工，比如你想要的薪资是否符合公司的标准

力哥七步定投策略

Posted on 2017-10-31 In 理财小知识 Word count in article: 1.3k Reading time ≈ 5 mins.
  1. 4大原则：
    1. 长期坚持定期不定额投资
    2. 长期坚持止盈不止损投资
    3. 只问贵贱，不看趋势
    4. 越跌越买，月涨越卖
  2. 5中投资策略
    1. 全天候配置策略
    2. 低估值择时轮动策略
    3. 价值平均策略
    4. 网格交易策略
    5. 动态再平衡策略
  3. 7个具体步骤
    1. 完成风险偏好测试
    2. 全面统计家庭财务信息，搞清楚有多少钱可以投资于金融资产
    3. 做好家庭资产配置方案
      1. 提取生活紧急备用金（3-6个月的生活支出放到货基中） ---现金流规划
      2. 购置适合的保险的产品： --- 家庭保障规划
        1. 消费险
        2. 储蓄型保险
      3. 将钱分为长期和短期（3年内） --- 消费规划和储蓄规划
    4. 设计存量资金的定投计划
      1. 设计定投阶段
      2. 布局定投阶段
      3. 维持定投阶段
      4. 存量资金定投计划策略
        1. 经济景气时积极进取，不景气时保守
        2. 经济周期
          1. 复苏阶段： 经济上行， 通胀（CPI）下行 增持股票
          2. 经济过热阶段： 经济上行，通胀上行 增持股票，大宗商品，减持债券、现金
          3. 滞涨阶段： 经济下行，通胀上行
            1. 减持各类资产，现金为王
          4. 衰退阶段： 经济下行，通胀下行
            1. 增持债券 美林时钟投资理论，按经济周期投资
        3. 指数估值
          1. PE 市盈率
          2. PB 市净率
          3. 绝对估值：市盈率超过50倍的不碰
          4. 相对估值：以自身的估值作为依据
          5. 市盈率分位点低于40%时开始定投，同时参考市净率的分位点
        4. 几个重要
          1. 存量资金定投布局要在一年内完成，资金量较少的建议每月定投一次，钱越多定投频率越高。
          2. 更推荐定投场内指数基金
    5. 在一年内完成定投布局 1. 低位多投，高位少投 2. 价值平均策略：每月完成定投后的市值是目标市值，而不是每月定投固定资金（目标市值恒定），例如：计划每月定投1万月，当第二个月定投时，持有基金的市值已达到了1万2千元，则第二月的定投金额改为8000，使市值达到2万月即可，如果第二月定投时市值降为8000元，则第二月的定投金额改为1万2千元。如果第二月定投时，市值已达到2万1千元，则要卖掉1千元市值的基金 3. 提高定投额标准 PE分位点 < 20% ,基准定投额提高1.5 PE分位点 < 15% ,基准定投额提高2 PE分位点 < 10% ,基准定投额提高2.5 PE分位点 < 5% ,基准定投额提高3 4. 减少定投额，抛售定投基金 
        当PE的分位点到达40即合理估值的区间时就要考虑减少定投或停止定投了。
    6. 对提前完成定投的基金进行网格收割（维持定投的操作）
      1. 网格交易操作
        1. 名词说明
        2. 网格交易的两个缺点：
          1. 熊市中可能买无可买
          2. 如果遇到大牛市，可能提前卖光
        3. 适用于震荡行情（与二八轮动恰恰相反）
      2. 当定投市值达到预定目标时，可以利用网格交易法则进行操作
        1. 从历史回测来看，当单边的熊市或牛市中网格可以放大一点
        2. 震荡行情中，网格就应该设置的小一点
        3. 最大的网格设置是20%，最小的网格设置是3%
        4. 上涨时的收割网格设置的要比下跌的时的补仓网格设置的要大一些，从而获得更大的利润空间
    7. 设置增量资金定投计划（要以年为周期）
      1. 全天候配置策略+低估值择时策略

    合理的资产配置+坚持科学的定投策略=永恒不变的投资王道

#定义一个class

class Demo:
    #定义一个私有变量
    __sex = "男"
    #类似于构造函数
    def __init__(self):
        self.name = "cruise";
    #定义私有方法
    def __getInfo(self):
        print("name:"+self.name+";地址：")
    #定义静态函数
    @staticmethod
    def staticMethod():
        print("this is a static method")

demo = Demo()
#调用私有方法
demo._Demo__getInfo();
Demo.staticMethod();
#使用id函数查看变量的唯一值
print(id(demo))
#调用私有属性
print(demo._Demo__sex)

  1. with...as.. 使处理异常更加方便

# -*- coding: utf-8 -*-
class fileLoader:
    def __init__(self,path):
        print("init")
        self.path = path;
    def __enter__(self):
        print("enter")
        self.file = open(self.path,"r");
        return  self
    def __exit__(self, exc_type, exc_val, exc_tb):
        print("exit")
        print(str(exc_type)+"::::::"+str(exc_val)+":::::"+str(exc_tb))
        # return True

    def showContext(self):
        print(self.file.read())

# with fileLoader("./test.txt") as fr:
#     fr.showContext()

with fileLoader("./classExample.py") as fr:
    fr.showContext()

使用with语句是执行顺序是init,enter,exit，如果程序在执行enter方法是出错，则错误的类型、示例、堆栈信息都会传到exit方法中（见exit方法参数）；如果exit方法最后返回True，则错误信息不会抛出；如果程序执行过程中没有抛出错误则exit方法中收到的异常对象为None

基金定投

Posted on 2017-10-08 In 理财小知识 Word count in article: 501 Reading time ≈ 2 mins.
  1. 定投的基本概念
    1. 定期定额投资
  2. 定投的好处：通过定投，不停的分批次小额买入基金，使基金的收益曲线更接近宏观经济的走势，收益曲线更加平滑
  3. 基金定投是使剧烈波动的基金收益变的平滑，因此波动比较的小的基金，如债券型基金，货币基金不适合做定投
  4. 定投优先选择指数基金
  5. 基金定投只是放弃了短期择时，而不能放弃牛熊大趋势的择时。
  6. 定期变额策略定投：
    1. 定投金额的变化规则
    2. 金额变化，操作起来更麻烦
  7. 定投频率越高，曲线越平滑
  8. 智能定投：
    1. 为实现越跌投越多的策略，智能定投运用的策略包括：均线策略，风险偏好策略，动态再平衡策略
    2. 均线策略:利用均线设置变额的梯度标准，均线低于目标均线多少比例，定投金额就增加多少比例；
      1. 蚂蚁财富的慧定投就是这种策略；
      2. 缺点：
        1. 没有充分考虑到定投开始时指数点位因素
        2. 没有解决止盈问题
    3. 风险偏好型策略智投：
      1. 根据投资者的风险承受能力和投资时长智能匹配基金类别和配置比例
      2. 招商银行的摩羯智投使用这种策略
      3. 缺点：
        1. 门槛较高
        2. 推送给投资者的主要是主动型基金
    4. 动态再平衡策略
      1. 选择关联度不大或负相关的基金构建基金组合
      2. 蛋卷基金的蛋定投
      3. 二八轮动不适合来回震动行情

    4. 指数基金比主动型基金更适合装死策略
  3. 换仓策略
    1. 换到看好的基金中
  4. 止损策略
    1. 割肉离场

场内、场外基金开户及股票行情

Posted on 2017-09-24 Edited on 2017-10-08 In 理财小知识 Word count in article: 1.3k Reading time ≈ 5 mins.

一、场外基金

  1. 不能在交易所上市交易的基金
  2. 直销、代销
  3. 直销：到基金公司官网注册开户，费用较低
  4. 代销：天天基金网，蛋卷基金
  5. 分红方式：现金分红、红利再投资。
  6. 货币基金统一都是红利再投资方式

二、 场内基金

  1. 手续费低、灵活、有一些套利玩法，无法设置定投
  2. 券商开户一般会开两个账户，一个是证券账户：沪A账户/深A账户；另一个是资金账户：用来对接银行存管账户。
  3. 银证转账，一般是在交易日的8:00-16:00
  4. 场内基金的买卖操作，要在券商交易系统中股票栏中操作，如果想尽快成交那么在卖出时选择买一价，买入时尽量选择卖一价。
  5. 场内基金分红方式默认现金分红且不能修改

三、股票行情软件

  1. K线图,又称作日本蜡烛线，其中实心的线叫做阴线，国内一般用绿色表示（海外用红色），空心的叫做阳线，国内用红色表示（为来显示效果好看，股票软件中有时会用实线表示，海外用绿色），所以有叫做的阴阳线；
  2. K线上方的细线叫做上影线，下方的细线叫做下引线
  3. K线中记录了开盘价，收盘价，当日最高价、当日最低价;当收盘价高于开盘价时，为阳线，上影线的最高点表示当日到达的最高价，下影线最低表示当日的最低价。
  4. 几种常见的K线
    1. 如果当天的开盘价或者收盘价就是当天的最高价那么就没有上影线，俗称光头
    2. 如果当天的开盘价或者收盘价时当天的最低价那么就没有下影线，俗称光脚
    3. 如果开盘价和收盘价相等那么柱子就会变成一条横线，加上上下影线，就是个十字型，叫做十字星，如果上下影线都很长，那么就叫做大十字星，反之叫做小十字星，如果只有下影线没有上影线就叫做T型光头的十字星，反过来，如果没有下影线只有上影线那么就叫做 倒T型光头十字星
    4. 如果当天开盘价，收盘价相等，并且没有什么波动（没有上影线和下影线）那么就叫做一字线
  5. 涨跌幅波动
    1. 小幅波动：涨跌幅0~1.5%
    2. 中幅破洞：涨跌幅1.5%~5%
    3. 大幅波动： 涨跌幅5%~10%
  6. K线的复权
    1. 向前复权：保持现有价格不变，将以前的价格折算响应缩减
    2. 向后复权：保持先前的价格不变，将除权后价格增加修复回去
  7. 移动平均线，简称均线（moving average，MA），如MA10就是指包含当天收盘价在内的20天的收盘价的平均值。
    1. 日均线指标：MA5，MA10 短期均线指标
    2. 季均线指标：MA30，MA60中期均线指标
    3. 年均线指标：MA120，MA250 长期均线指标
    4. 一般MA250就是年线
    5. 也可以自己设定均线。
    6. 短期均线超过中长期均线叫做金叉，意味着趋势向上，反之则叫做死叉
  9. 成交量：在k线图和均线图下方的柱状图
    1. 成交量均线（mavol）
    2. 技术分析研究股票的量、价、时、空
    3. 成交量决定了股票的流动性
  10. 涨幅：收盘价相对于前一个交易日收盘价的上涨幅度
  11. 振幅：当天最高价和最低价的差额相对于昨天的收盘价的百分比，体现的是股票或基金的活跃性
  12. 分时走势：黄色的线表示成交价格，白色的线表示成交量。
  13. 快捷键：
    1. F5切换分时走势和技术分析
    2. F6查看自选股
    3. F3快速查看上证指数
    4. F4快速查看深证成指
  14. 基金小白要避免的误区
    1. 把握时机，不能为了长期投资而长期投资，要学会止损，止盈
    2. 净值不是越便宜越好
      1. 净值低原因1.这只基金是新基金
      2. 本身业绩就差
      3. 老基金刻意通过大幅分红和拆分降低净值，有可能会摊薄老基金持有人的利益
      4. 是不是分红越多的基金越好？当然不是

docker安装zookeeper集群

Posted on 2017-09-20 Edited on 2021-09-04 Word count in article: 853 Reading time ≈ 3 mins.
  1. 更改docker镜像源 vi /etc/default/docker 
    DOCKER_OPTS="--registry-mirror=http://aad0405c.m.daocloud.io"
  2. 拉去zookeeper镜像 
    docker pull  garland/zookeeper //不要使用官方的zookeeper；garland/zookeeper镜像有些额外配置，对新手来说非常友好

    docker run -d \
--name=zk1 \
--net=host \
-e SERVER_ID=1 \
-e ADDITIONAL_ZOOKEEPER_1=server.1=localhost:2888:3888 \
-e ADDITIONAL_ZOOKEEPER_2=server.2=localhost:2889:3889 \
-e ADDITIONAL_ZOOKEEPER_3=server.3=localhost:2890:3890 \
-e ADDITIONAL_ZOOKEEPER_4=clientPort=2181 \
garland/zookeeper

-e ADDITIONAL_ZOOKEEPER_2=server.2=localhost:2889:3889 \

garland/zookeeper

    --net=host指令指定改容器使用host网络模式，那么这个容器将不会获得一个独立的Network Namespace，而是和宿主机共用一个Network Namespace。容器将不会虚拟出自己的网卡，配置自己的IP等，而是使用使用宿主机的IP和端口，不用任何NAT转换，就如直接跑在宿主机中的进程一样。但是，容器的其他方面，如文件系统、进程列表等还是和宿主机隔离的。

-e 指定镜像创建完毕执行的命令的参数， garland/zookeeper镜像创建完容器会执行/opt/run.sh，-e指定的参数就是run.sh运行时的参数，run.sh内容如下：

#!/bin/sh
ZOO_CFG="/opt/zookeeper/conf/zoo.cfg"
# Output server ID
echo "server id (myid): ${SERVER_ID}"
echo "${SERVER_ID}" > /tmp/zookeeper/myid
# Add additional ZooKeeper servers into the zoo.cfg file
echo "${ADDITIONAL_ZOOKEEPER_1}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_2}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_3}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_4}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_5}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_6}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_7}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_8}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_9}" >> ${ZOO_CFG}
echo "${ADDITIONAL_ZOOKEEPER_10}" >> ${ZOO_CFG}
# Start Zookeeper
/opt/zookeeper/bin/zkServer.sh start-foreground

  1. 由于是使用host网络模式，所以在连接zookeeper时可以直接使用宿主机的IP，如

    zkCli.cmd -server 192.168.74.131:2181
   zkCli.cmd -server 192.168.74.131:2182
   zkCli.cmd -server 192.168.74.131:2183

  2. 附上garland/zookeeper Darkfile文件

    # DOCKER-VERSION 1.0.1
# VERSION        0.5
# SOURCE         https://github.com/jplock/docker-zookeeper 
FROM debian:jessie
MAINTAINER Justin Plock <justin@plock.net>
RUN apt-get update && apt-get install -y openjdk-7-jre-headless wget
RUN wget -q -O - http://apache.mirrors.pair.com/zookeeper/zookeeper-3.4.6/zookeeper-3.4.6.tar.gz | tar -xzf - -C /opt \
&& mv /opt/zookeeper-3.4.6 /opt/zookeeper \
&& cp /opt/zookeeper/conf/zoo_sample.cfg /opt/zookeeper/conf/zoo.cfg \
&& mkdir -p /tmp/zookeeper
ENV JAVA_HOME /usr/lib/jvm/java-7-openjdk-amd64
ADD ./run.sh /opt/run.sh
RUN chmod 777 /opt/run.sh
EXPOSE 2181 2888 3888
WORKDIR /opt/zookeeper
VOLUME ["/opt/zookeeper/conf", "/tmp/zookeeper"]
CMD ["/opt/run.sh"]

  3. 附上一些常用命令

    1. 停用全部运行中的容器: docker stop $(docker ps -q)
    2. 删除全部容器： docker rm $(docker ps -aq)
    3. 一条命令实现停用并删除容器： docker stop $(docker ps -q) & docker rm $(docker ps -aq)
    4. 将宿主机2181端口和容器docker端口对应，并将容器命名为zk1 docker run -d -p 2181:2181 --name zk1 zookeeper
    5. 使用zkCli.cmd连接该zookeeper验证是否部署成功 zkCli.cmd -server 宿主机IP:2181 docker run -it -link=zk1:zkCli zookeeper zkCli.sh -server 192.168.74.131
    6. docker logs -t --tail=20 容器ID 查看容器日志
